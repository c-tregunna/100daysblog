Day 5

Time to cover data types. It appears this has updated since I originally started learning JS and currently (2021) there are 7 data types in JS. Let's have a look at them and what they are used for:

  1. Number: Floating point numbers, always have a decimal point even if you do not see it - Used for decimal and integer
  2. String: A sequence of characters, used for text. Always need to be in quotes, either ' ' or " "
  3. Boolean: Logical type, only ever true or false. Used for making decisions
  4. Undefined: Value taken by a variable that is not yet defined
  5. Null: Also means an empty value. Will show as an object using typeof. This is a known bug in JS
  6. Symbol: Came along in ES2015. A value that is unique and cannot be changed. Huh??
  7. BigInt: Came along in ES2020. Large integers that a Number type can't hold. WHAT??

So there are a couple of new ones here to me, but sure these will be covered as I work through the course. The main thing here is you will usually use the first 3 data types in your code.

What does each data type look like?

In the console.log() you will see typeof. Using this will show you in the console the data type you have defined. The great thing with JS is it will auto define a data type when stored in a variable. WHOOP!!
